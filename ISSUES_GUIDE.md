# Issues Guide for Shahira Code

When creating an issue, please include the following information to help us resolve it quickly:

## For Bug Reports
- **Describe the bug**: What happened? What did you expect?
- **Steps to reproduce**: List the steps to reproduce the issue.
- **Screenshots**: Attach if relevant.
- **Environment**: OS, browser, device, etc.
- **Error messages/logs**: Paste any relevant output.

## For Feature Requests
- **Describe the feature**: What do you want to see?
- **Use case**: Why is this feature important?
- **Alternatives**: Any workarounds you use?

## For Security Issues
- **Do not post sensitive vulnerabilities publicly.**
- Email us directly at [your-email@example.com] for responsible disclosure.

## For General Questions
- Use the Discussions tab or open an issue with the `question` label.

---

**Tips for Effective Issues:**
- Be clear and concise.
- Search existing issues before posting.
- Use labels to categorize your issue (bug, enhancement, question, etc.).
- Be respectful and constructive.
